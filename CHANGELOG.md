# 변경 이력 (Changelog)

이 문서는 AI 할 일 관리 서비스의 주요 변경사항을 기록합니다.

---

## [1.2.0] - 2026-01-17

### 새 기능
- **설정 페이지**: 테마(라이트/다크/시스템), 언어, 계정 삭제 기능
- **이메일 알림**: 중요도 높은 할 일 24시간 이상 지연 시 이메일 알림
- **프로필 페이지**: 사용자 정보 확인, 이름 변경, 비밀번호 변경
- **다크 모드**: next-themes 기반 테마 시스템 도입

### 개선
- 로그아웃 후 로딩 스피너 무한 회전 버그 수정
- README.md 전면 업데이트 (구현 기능 반영)
- 프로젝트 규칙 문서 업데이트

### 인프라
- Resend 이메일 서비스 연동
- Vercel Cron Jobs 설정 (매일 오전 9시 알림 체크)
- favicon 및 Apple 아이콘 업데이트

## [1.1.0] - 2026-01-16

### 새 기능
- **AI 자연어 파싱**: 자연어로 할 일 입력 시 자동으로 구조화된 데이터로 변환
- **AI 요약 및 분석**: 일간/주간 생산성 분석, 완료율, 개선 제안 제공
- **할 일 CRUD**: 생성, 조회, 수정, 삭제 기능 (Supabase 연동)
- **검색 및 필터**: 제목 검색, 우선순위/상태/카테고리 필터링
- **정렬 기능**: 생성일, 마감일, 우선순위 기준 정렬

### 인증
- Supabase 기반 이메일/비밀번호 인증
- 이메일 인증 (회원가입 시)
- 비밀번호 재설정

### UI/UX
- 반응형 대시보드 레이아웃
- Modern Productivity 컬러 팔레트 적용
- shadcn/ui 컴포넌트 시스템
- 토스트 알림 (sonner)
